Gemini CLI

Explicação detalhada do Dockerfile para o antigo assistente de linha de comando do Google.

Voltar

Este guia descreve como o container Docker está estruturado para rodar o Gemini CLI de forma reprodutível e isolada.

Dockerfile Original

FROM ubuntu

ENV DEBIAN_FRONTEND=noninteractive

RUN apt-get update && apt-get install -y \
    curl \
    gnupg \
    && curl -fsSL https://deb.nodesource.com/setup_20.x | bash - \
    && apt-get install -y nodejs \
    && rm -rf /var/lib/apt/lists/*

RUN npm install -g @google/gemini-cli

WORKDIR /workspace

CMD ["bash"]

Explicação Passo a Passo

1. Imagem Base

FROM ubuntu: Utiliza a distribuição Ubuntu Linux mais recente como base para a montagem de todo o sistema do container.

2. Configuração de Ambiente

ENV DEBIAN_FRONTEND=noninteractive: Instrui o gerenciador de pacotes do Debian/Ubuntu a não abrir menus e janelas interativas na instalação.

3. Instalação do Node.js

Os comandos de build executam a preparação e limpeza da plataforma:

  • Atualiza os repositórios (apt-get update).
  • Instala utilitários essenciais: curl e gnupg.
  • Baixa e configura o instalador do Node.js na versão LTS 20.x.
  • Instala o runtime nodejs e remove resíduos de cache para otimização de espaço.
4. Instalação Global da CLI

RUN npm install -g @google/gemini-cli: Instala de forma global no sistema o pacote oficial da interface de terminal do Gemini.

5. Diretório de Execução

WORKDIR /workspace: Define a pasta default de trabalho do container para montagem de volumes.

6. Inicializador

CMD ["bash"]: Garante que um shell interativo Bash seja aberto automaticamente ao rodar o container.

Comandos de Execução

Instruções rápidas para construir e testar este ambiente legado:

Construir Imagem
docker build -t gemini-cli ./gemini-cli
Executar via Compose
docker compose -f docker-compose-gemini.yml run --rm gemini

Status do Gemini CLI: Descontinuado

O Google decidiu descontinuar o pacote @google/gemini-cli. O novo padrão oficial de desenvolvimento e assistência autônoma é o Antigravity CLI. Recomendamos acessar o guia interativo do Antigravity CLI para usar a versão moderna do agente.
