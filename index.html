Como Executar em Minutos

Você pode levantar cada ambiente rapidamente copiando os comandos Docker prontos.

1
Certifique-se de ter o Docker e Docker Compose instalados.
2
Clone este repositório em sua máquina.
3
Execute o comando do agente desejado e comece a interagir!
Construir Imagem Gemini CLI
docker build -t gemini-cli .
Executar Gemini CLI de forma interativa
docker run -it --rm -v $(pwd):/workspace gemini-cli
Construir Imagem Antigravity
docker build -t antigravity-cli .
Executar Antigravity de forma interativa
docker run -it --rm -v $(pwd):/workspace antigravity-cli
Construir Imagem Claude Code
docker build -t claude-code .
Executar Claude Code de forma interativa
docker run -it --rm -v $(pwd):/workspace claude-code

Por que rodar Agentes em Docker?

Isolamento Total

Agentes executam códigos e dependências que poderiam interferir com a sua máquina local. O Docker garante que nada saia do container.

Reprodutibilidade

Esqueça o clássico "na minha máquina funciona". Qualquer pessoa que baixar o repositório terá a mesma versão exata do Node/Ubuntu e pacotes.

Configuração Zero

Toda a instalação e compilação de binários ocorrem nos passos do Dockerfile. Você só precisa executar os containers prontos.
